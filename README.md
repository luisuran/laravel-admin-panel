# laravel-admin-panel
Simple laravel project to manage companies and their employees

### Requirements
- php >= 8.0

### Instalation
1. `composer install`
2. Rename .env.example with .env
3. `./vendor/bin/sail up`
4. `./vendor/bin/sail artisan key:generate`
